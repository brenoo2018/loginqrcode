
<h2 align="center">
 App Login + QRCode
</h2>
Um aplicativo em react native com login de autenticação utilizando context api persistindo os dados no async storage e qr code para leitura de dados.

### Ferramentas utilizadas:
- React Navigation;
- Context API;
- Async Storage;
- React Native QRCode SVG;
- Styled Components;
- Axios;
- Formik;
